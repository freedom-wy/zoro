# zoro  
罗罗亚·佐罗（ロロノア・ゾロ）是日本漫画家尾田荣一郎所创作的少年漫画《航海王》中的虚构角色，是该作品的中的一位主要角色，是主角路飞的第一位伙伴，草帽海贼团的战斗员。他立志成为世界最强的剑士，他以双手持刀、嘴上再咬一把刀的三刀流战斗风格闻名，一名路痴，路人的指示下也会走错，团员中第二号人物。
